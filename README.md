# Bank Reviews Week 2

This repo contains the Week 2 interim work for the Customer Experience Analytics challenge.

## Reproduce
1. Install dependencies:
   pip install -r requirements.txt

2. Scrape (replace app ids in src/scrape_reviews.py):
   python src/scrape_reviews.py

3. Preprocess:
   python src/preprocess.py

4. Create interim plots:
   python src/plots_for_interim.py

Interim report: reports/interim_report.pdf
